UVN Facial Rig

Project Status Concluded
Project Duration 4 Months
Software Used Maya
Languages Used Bifrost Nodes
Project Role(s) Sole Developer

Overview

As a personal project during the 2023 fall semester, I reconstructed the UVN facial rig used in Blue Sky Studios’ Peanuts Movie. The final version of the rig is made with the Bifrost node system, which I studied online during the latter half of the semester.

Project Goals

  • Recreate the facial rig used in Blue Sky Studios' Peanuts Movie

  • Become familiar with technical tools used in the animation industry

  • Develop familiarity with node-based visual programming
