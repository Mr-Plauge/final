Brachiation Game Mechanic

Project Status Concluded
Project Duration 3 months
Software Used Visual Studio, Unity, Maya
Languages Used C#
Project Role(s) Lead Developer

Overview

My objective for this project was to replicate the arboreal movements of primates as a video game mechanic. To capture the swinging motions necessary for this form of locomotion, I developed a player movement system built on swiping the mouse back and forth to build momentum.

Project Goals

  • Replicate the brachiation patterns exhibited by arboreal primates

  • Develop familiarity with physics-based procedural animation

  • Design a mouse-based player control system that captures the feeling of swinging
